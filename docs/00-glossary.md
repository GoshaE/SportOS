# 00. Глоссарий терминов SportOS

> Единый справочник терминов проекта. При первом упоминании в любом документе термин выделяется *курсивом*.

---

## Общие термины

| Термин | EN | Определение |
|---|---|---|
| **Мероприятие** | Event | Соревнование, объединяющее несколько дисциплин (напр. «Чемпионат Урала 2026») |
| **Дисциплина** | Discipline | Конкретный вид соревнования внутри мероприятия (напр. «Скиджоринг 5 км») |
| **Гонка** | Race | Один заезд в рамках дисциплины. В многодневных мероприятиях у дисциплины несколько гонок (Day 1, Day 2) |
| **Трасса** | Course / Track | Физический маршрут соревнования, описанный GPX-треком и дистанцией |
| **Круг** | Lap | Один проход по замкнутой трассе. Дисциплина может включать N кругов |

## Участники

| Термин | EN | Определение |
|---|---|---|
| **Атлет** | Athlete | Спортсмен-участник. Имеет глобальный профиль в базе |
| **Заявка** | Entry | Связка «Атлет + Дисциплина + Собаки». Одна заявка = одно участие |
| **BIB** | BIB / Bib Number | Стартовый номер атлета. Назначается при жеребьёвке |
| **Упряжка** | Team (dogs) | Набор собак, привязанных к заявке |
| **Категория** | Category | Группировка для протокола результатов (М/Ж, CEC/OPEN, юниоры/ветераны) |
| **Стартовый поток** | Start Wave | Группа атлетов, стартующих вместе или последовательно. Может совпадать с категорией или нет |

## Хронометраж

| Термин | EN | Определение |
|---|---|---|
| **Отсечка / Метка** | TimeMark | Зафиксированный момент времени (нажатие кнопки судьёй или сигнал от чипа) |
| **Очередь меток** | Timestamp Queue | Список неназначенных отсечек, ожидающих привязки к BIB |
| **Мастер-таймер** | Master Timer | Устройство с официальным временем. При потере — failover на контрольное |
| **Контрольный таймер** | Control Timer | Резервное устройство, дублирующее отсечки мастера |
| **Clock Offset** | Clock Offset | Разница между системными часами устройства и эталоном (GPS/NTP), в миллисекундах |
| **Split-time** | Split | Промежуточное время на момент прохождения круга или контрольной точки |
| **Net Time** | Net Time | Чистое время прохождения дистанции (финиш − старт − штрафы) |
| **Gross Time** | Gross Time | Полное время (финиш − старт, без вычета штрафов) |

## Старты

| Термин | EN | Определение |
|---|---|---|
| **Масс-старт** | Mass Start | Все атлеты стартуют одновременно по сигналу |
| **Раздельный старт** | Individual / Interval Start | Атлеты стартуют по одному с интервалом |
| **Гундерсен** | Pursuit / Handicap Start | Старт по отставанию от лидера предыдущего этапа |
| **DNS** | Did Not Start | Не стартовал |
| **DNF** | Did Not Finish | Не финишировал (сошёл) |
| **DSQ** | Disqualified | Дисквалифицирован |

## Роли

| Термин | EN | Определение |
|---|---|---|
| **Администратор** | Admin | Полный доступ, управление мероприятием, финансами, жеребьёвкой |
| **Главный судья** | Head Judge | Утверждение результатов, разрешение конфликтов, вторая позиция в цепочке наследования |
| **Судья (финиш)** | Finish Judge | Работа с мастер-таймером на финишной линии |
| **Стартёр** | Starter | Управление порядком на стартовой зоне, обратный отсчёт |
| **Маршал** | Marshal / Course Marshal | Волонтёр на трассе, отмечает прохождение контрольных точек, фиксирует нарушения |
| **Ветеринар** | Vet | Проверка допуска собак (вакцинации, состояние) |
| **Диктор** | Announcer | Комментирование соревнований, мониторинг live-результатов |

## Синхронизация

| Термин | EN | Определение |
|---|---|---|
| **P2P Mesh** | Peer-to-Peer Mesh | Децентрализованная сеть из устройств судей, синхронизирующихся напрямую |
| **CRDT** | Conflict-free Replicated Data Type | Структура данных, автоматически разрешающая конфликты при распределённой записи |
| **LWW** | Last-Writer-Wins | Стратегия разрешения конфликтов: побеждает последняя запись |
| **Векторные часы** | Vector Clock | Механизм отслеживания причинно-следственных связей между событиями на разных устройствах |
| **Tombstone** | Tombstone | Запись soft-delete в CRDT; маркер «удалено», не удаляется физически |
| **Split-brain** | Split-brain | Ситуация, когда часть mesh-сети теряет связь и работает автономно |
| **Авторитетное устройство** | Authority Device | Устройство с правом финального утверждения результатов |

## Статусы

| Термин | EN | Определение |
|---|---|---|
| **Provisional** | Provisional | Предварительные результаты, могут измениться (grace-period) |
| **Verified** | Verified | Все маршалы синхронизированы, данные полные |
| **Official** | Official | Утверждены главным судьёй, окончательные |

## Ездовой спорт

| Термин | EN | Определение |
|---|---|---|
| **Mushing** | Mushing | Ездовой спорт — гонки на собачьих упряжках |
| **Скиджоринг** | Skijoring | Лыжник + 1 собака |
| **Каникросс** | Canicross | Бегун + 1 собака |
| **Пулка** | Pulka | Лыжник + 1-2 собаки с санями-пулкой |
| **Нарта** | Sled | Сани для упряжки собак (нарта-2, нарта-4, нарта-6, нарта-8) |
| **CEC** | CEC | Класс сертифицированных спортсменов |
| **OPEN** | OPEN | Открытый класс (любительский) |
| **Чип** | Chip / Microchip | Электронный идентификатор, вживлённый собаке |
