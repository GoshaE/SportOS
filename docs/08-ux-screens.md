## 1. Общие UX-принципы

| Принцип | Реализация |
|---|---|
| **Thumb-friendly** | Ключевые кнопки — внизу экрана, зона большого пальца |
| **Работа в перчатках** | Крупные touch-targets (min 48dp), отсутствие мелких элементов |
| **Wakelock** | Экран не гаснет в Ops-режимах |
| **Вибрация + звук** | Тактильная и звуковая обратная связь при отсечке |
| **Landscape опц.** | Ops-экраны поддерживают горизонтальный режим |
| **Responsive** | Адаптация под телефон и планшет |
| **Одно приложение** | Один бинарник, интерфейс адаптируется к роли |

## 2. Экран финиша

Ключевой экран системы — основной инструмент судьи.

### Компоновка

```
┌──────────────────────────────────┐
│ 🏁 Sprint 5km │ ⏱ 01:23:45      │ ← Инфо-панель
│ Финиш: 12/35  │ 🟢 Мастер       │    (компактная)
├──────────────────────────────────┤
│                                  │
│  Список финишировавших (scroll): │
│                                  │
│  1. 00:38:12  BIB 07 Петров ✅   │ ← назначен
│  2. 00:39:45  BIB 24 Иванов ✅   │ ← назначен
│  3. 00:41:02  ────────────  ⚠️   │ ← НЕ назначен
│  4. 00:41:33  ────────────  ⚠️   │ ← НЕ назначен
│  5. 00:42:15  ────────────  ⚠️   │ ← только что
│                                  │
│          ▼ листается ▼           │
│                                  │
├──────────────────────────────────┤
│  ╔══════════════════════════════╗│
│  ║        🔴 ОТСЕЧКА           ║│ ← прижата внизу
│  ╚══════════════════════════════╝│
└──────────────────────────────────┘
```

### Элементы

| Зона | Элемент | Описание |
|---|---|---|
| **Верх** | Инфо-панель | Дисциплина, бегущее время, прогресс, статус sync, роль (мастер/контрольный) |
| **Центр** | Список отсечек | Хронологический порядок (первый финиш сверху). Скроллируемый |
| **Низ** | Большая кнопка ОТСЕЧКА | Прижата к нижнему краю, всегда видна. ~20% высоты экрана |

### Поведение

1. **Тап на ОТСЕЧКА** → вибрация + звук + новая строка в списке (без BIB, ⚠️)
2. **Тап на неназначенную строку** → модальное окно с плитками BIB
3. **Свайп влево** → удаление метки (для пустых — мгновенно, для назначенных — подтверждение)
4. **Long press** → контекстное меню: редактировать время, сменить BIB, снять BIB

### Модальное окно назначения BIB

```
┌──────────────────────────────────┐
│ 🔍 Поиск по номеру / фамилии    │
├──────────────────────────────────┤
│ Chips: [Все] [Скиджоринг] [Нарта] │ ← фильтр по дисциплине
├──────────────────────────────────┤
│                                  │
│  [07 Петров] [12 Сидоров] [24..] │
│   Скидж.      Скидж.      Нарта │ ← дисциплина на плитке
│  [31 Козлов] [42 Морозов] [55..] │
│   Нарта       Нарта       Пулка │  ← 3 колонки плиток
│                                  │
├──── выбранные (серые) ───────────┤
│  [03 Иванов ✓] [15 Алексеев ✓]  │ ← уже финишировали
│                                  │
└──────────────────────────────────┘
```

**Правила фильтрации плиток**:
- Показывать **только стартовавших** (`status = started`)
- Атлеты, которые **не стартовали**, не отображаются
- Уже **финишировавшие** — внизу, серые, неактивные
- Тап на плитку → BIB назначен → модалка закрывается

## 3. Экран стартёра

### Компоновка

```
┌──────────────────────────────────┐
│ 🏁 Sprint 5km │ ⏱ 10:04:45      │
│ Интервал: 30с  │ Осталось: 23    │
├──────────────────────────────────┤
│                                  │
│  ⏰ Следующий старт через:       │
│        00:12                     │ ← обратный отсчёт
│  BIB 24 — Иванов А.А.           │
│                                  │
├──────────────────────────────────┤
│  Очередь:                        │
│  ✅ BIB 07 Петров — ушёл 10:04:00│
│  ⏳ BIB 24 Иванов — 10:04:30    │ ← текущий
│  ⏳ BIB 31 Козлов — 10:05:00    │
│  ⏳ BIB 42 Морозов — 10:05:30   │
├──────────────────────────────────┤
│  [DNS]          [Ушёл ✅]        │ ← внизу
└──────────────────────────────────┘
```

### Поведение (раздельный старт)

- Старт **автоматический по времени** (обратный отсчёт)
- Стартёр нажимает **«Ушёл»** — подтверждение. Или **«DNS»** — не стартовал
- Если стартёр ничего не нажал и время прошло — атлет считается стартовавшим автоматически
- **Отмена DNS**: если стартёр ошибся — long press на атлета → «Отменить DNS»

### Принудительный ранний старт

- Тап на атлета в очереди (не текущего) → меню:
  - **«Стартовать сейчас»** → ActualStartTime = текущий момент
  - **«Привязать к BIB X»** → ActualStartTime = время старта BIB X
- Остальные атлеты продолжают свой порядок без изменений
- Запись в *Audit Log*

### Поведение (масс-старт)

- Большая кнопка **«GUN START»** — запускает всех одновременно
- При нажатии: подтверждение 5 секунд («Отменить старт?»)
- Повторный «GUN» — `ZeroTime` пересчитывается

## 4. Экран маршала

### Компоновка — основной экран из плиток

```
┌──────────────────────────────────┐
│ 🚩 Checkpoint: 3 км │ 🟢 Online │
│ Отмечено: 18/35                  │
├──────────────────────────────────┤
│                                  │
│  [07 Петров✓] [12 Сидоров] [24..]│
│  [31 Козлов ] [42 Морозов] [55..]│  ← 3 колонки
│  [63 Волков ] [77 Лебедев] [88..]│
│                                  │
│          ▼ листается ▼           │
│                                  │
├──────────────────────────────────┤
│  [⚖️ Зафиксировать нарушение]    │
│  [🆘 Экстренная ситуация]      │
└──────────────────────────────────┘
```

### Поведение

- **Тап на плитку** → атлет отмечен (✓ + время прохождения). Плитка меняет цвет
- **Повторный тап** → отмена отметки (с подтверждением)
- **Long press** → меню: «Зафиксировать нарушение», «Запрос DNF»
- Плитки фильтруются: **только стартовавшие**, неотмеченные — сверху
- **Фиксация нарушения**: текстовое описание + фото (опц.) → отправляется судье
- **🆘 SOS**: экстренный alert всем устройствам в mesh (время, checkpoint, описание)

## 5. Экран ветеринара

### Компоновка

```
┌──────────────────────────────────┐
│ 🩺 Ветконтроль │ Проверено: 8/25│
├──────────────────────────────────┤
│                                  │
│  Список собак:                   │
│  ⚠️ Rex (чип 123456) — Иванов   │
│     Вакц. до: 15.01.2026 ⏰     │
│  ⬜ Luna (чип 789012) — Петров  │
│     Вакц. до: 20.05.2026 ✅     │
│  ✅ Storm (чип 345678) — Козлов │
│     Допущен                      │
│                                  │
├──────────────────────────────────┤
│  [✅ Допустить]  [❌ Не допустить]│
└──────────────────────────────────┘
```

### Поведение

- Список собак с информацией о вакцинациях
- **Тап на собаку** → карточка с деталями (чип, владелец, вакцинации, фото)
- Два действия: **допустить** / **не допустить** (с причиной)
- **Grace-period** для вакцинации: конфигурируемый (напр. 30 дней). Администратор может override
- Двухэтапная проверка при замене собаки: автоматическая проверка чипа → подтверждение ветеринара

## 6. Экран диктора

### Компоновка

```
┌──────────────────────────────────┐
│ 🎙 Live │ Sprint 5km │ 🏃 23/35 │
├──────────────────────────────────┤
│                                  │
│  🏆 ТОП-5:                       │
│  1. BIB 07 Петров   00:38:12    │
│  2. BIB 24 Иванов   00:39:45 +1:33│
│  3. BIB 31 Козлов   00:41:02 +2:50│
│  4. BIB 42 Морозов  00:41:33    │
│  5. BIB 55 Волков   00:42:15    │
│                                  │
│  ⚡ Последний финиш:             │
│  BIB 55 Волков — 00:42:15 (5-е) │
│                                  │
│  📊 На трассе: 23 атлетов        │
│  ⏱ Ожидается: ~2 мин            │
│                                  │
├──────────────────────────────────┤
│  Подсказки:                      │
│  🔥 «BIB 55 вошёл в ТОП-5!»     │
│  ⏰ «Последний на трассе: BIB 88»│
└──────────────────────────────────┘
```

### Поведение

- **Тап на атлета** → карточка: ФИО, клуб, город, собаки, split-times по кругам, история стартов
- **Автоматические подсказки**: «Новый лидер!», «Вошёл в ТОП-3», «Последний на трассе»
- **Прогноз прибытия**: на основе средней скорости прохождения кругов
- **Обновление в реальном времени**: корректировки BIB, штрафы — всё видно мгновенно
