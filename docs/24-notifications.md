# 24. Система уведомлений

> Единый документ: каналы доставки, типы уведомлений, настройки, приоритеты.

---

## 1. Каналы доставки

| Канал | Стоимость | Офлайн | MVP | Описание |
|-------|-----------|:------:|:---:|----------|
| **Push (FCM/APNs)** | Бесплатно | ❌ | ✅ | Основной канал |
| **In-App (Inbox)** | Бесплатно | ✅ | ✅ | Список в табе 🔔, сохраняется локально |
| **Email** | Бесплатно* | ❌ | ✅ | Важные (результаты, регистрации) |
| **Telegram бот** | Бесплатно | ❌ | ✅ | Opt-in, см. [22-telegram-bot.md](22-telegram-bot.md) |
| **P2P Mesh** | Бесплатно | ✅ | ✅ | Только во время гонки, внутри mesh |

\* Email — через Firebase / Resend (бесплатный тир достаточен для MVP)

## 3. Настройки пользователя (PR6)

### Группы настроек

| Группа | Переключатели |
|--------|---------------|
| **Мероприятия** | Результаты, стартовые листы, расписание |
| **Клубы** | Заявки, взносы, мероприятия клуба |
| **Тренер** | Результаты воспитанников, PB, заявки |
| **Собаки** | Вакцинации, ветконтроль |
| **Маркетинг** | Новости, рекомендации (по умолчанию OFF) |

### Гранулярность по каналам

Пользователь настраивает **per-группа + per-канал**:

```
Мероприятия
  ├── Push       [✅]
  ├── Email      [✅]
  └── Telegram   [✅]
  
Клубы
  ├── Push       [✅]
  ├── Email      [❌]
  └── Telegram   [✅]
  
Маркетинг
  ├── Push       [❌]
  ├── Email      [❌]
  └── Telegram   [❌]
```

### Тихий режим

| Настройка | Описание |
|-----------|----------|
| Не беспокоить | Время с-по (например 23:00 — 07:00) |
| Исключения | 🔴 Критичные (Security Alert, SOS) доставляются всегда |

## 4. In-App Inbox (таб 🔔)

### Структура

```
┌─────────────────────────────────────┐
│ 🔔 Уведомления                      │
│ [Все] [Мероприятия] [Клубы] [Система]│
├─────────────────────────────────────┤
│ 🔴 Ночная гонка — результат: 3 место│
│    Скиджоринг 6км · 23:15 · сегодня │
│─────────────────────────────────────│
│ 🟡 BIB 42 назначен                  │
│    Чемпионат Урала · вчера           │
│─────────────────────────────────────│
│ 🟢 Ветконтроль Rex ✅                │
│    Нарта 10км · вчера                │
│─────────────────────────────────────│
│ 📩 Заявка в «Быстрые лапы» принята │
│    3 дня назад                       │
└─────────────────────────────────────┘
```

### Хранение

| Параметр | Значение |
|----------|----------|
| Локальное | SQLite/Isar — неограниченно |
| Sync | При подключении к облаку |
| Офлайн | Mesh-уведомления сохраняются локально, синхронизируются позже |
| Прочитано/Нет | Бейдж на табе 🔔 |
