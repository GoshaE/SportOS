# 22. Telegram-бот SportOS

## Назначение

Telegram-бот — бесплатный канал коммуникации между SportOS и пользователями.
Используется для верификации, уведомлений и быстрого доступа к данным.

---

## Функции бота

### 1. 🔐 Верификация и привязка аккаунта

| Сценарий | Описание |
|----------|----------|
| **Привязка результатов гостя** | Бот отправляет код подтверждения — пользователь вводит его в приложении |
| **Привязка Telegram к аккаунту SportOS** | `/start` → бот выдаёт одноразовый код → ввести в настройках приложения |
| **Двухфакторная авторизация (v2)** | Код через бота при входе с нового устройства |

#### Флоу привязки результатов гостя

    1. Пользователь в приложении нажимает «Привязать результат»
    2. Выбирает «Через Telegram»
    3. Приложение показывает: «Напишите боту /verify XXXX»
    4. Бот получает команду, проверяет код
    5. Бот отвечает: «✅ Код верный. Результат привязан!»
    6. Приложение получает callback и обновляет профиль

## Техническая архитектура

    SportOS Backend (Firebase / Supabase)
         │
         ├── Telegram Bot API (python-telegram-bot / Node.js)
         │      ├── Webhook: POST /telegram/webhook
         │      ├── Команды (/start, /verify, /results...)
         │      └── Push уведомления через sendMessage
         │
         └── Связка: users.telegramChatId ↔ userId

### Связка аккаунта

```
User {
  ...
  String? telegramChatId;     // null = не привязан
  String? telegramUsername;    // @username
  bool telegramNotifications; // true/false
}
```

### Безопасность

- Одноразовые коды (TTL 5 минут)
- Rate limiting (3 попытки в минуту)
- Привязка 1 Telegram ↔ 1 аккаунт SportOS
- Отвязка через настройки приложения
