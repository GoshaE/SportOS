Ключевые связи между блоками

переходы
🔐 Авторизация
A3 → Hub (H1) — после входа
Любой экран → A2 — мягкий триггер (регистрация на мероприятие)
QR-скан → R3 — прямой вход маршала
🏠 Хаб → Мероприятие
H3 → H4 — кнопка «Зарегистрироваться» (атлет)
H3 → E1 — кнопка «Управлять» (организатор)
H3 → RS1 — кнопка «Результаты» (все)
⚙️ Управление → Подготовка
E3 → P1 — кнопка «Жеребьёвка»
P1 → P2 — утверждение → стартовый лист
P1 → P3 — BIB назначение
🏁 Управление → Ops
E1 «Режим гонки» → R1-R5 — выбор роли
Ops внутри: R1↔R2↔R3↔R4↔R5 — переключение вкладок
📊 Ops → Результаты
R2 (финиш) → RS1 — автообновление live
Дисциплина завершена → RS1 Provisional
RS1 Official → RS4 — генерация дипломов
RS3 (вердикт) → RS1 — пересчёт (ScoringEngine)
🌐 Приложение → Web
P2 (стартовый лист) → W2 — публикация
RS1 (live) → W3 — WebSocket push
RS4 (дипломы) → PR4 — push-уведомление атлету
