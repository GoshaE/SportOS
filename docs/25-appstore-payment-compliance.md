# 25. Комплаенс оплат: App Store и Google Play

> Рекомендации по соответствию политикам Apple и Google при обработке платежей за мероприятия.

---

## 1. Ключевое правило

| Тип товара | IAP обязателен? | Комиссия |
|---|---|---|
| **Цифровой контент** (подписки, фичи, контент) | ✅ Да | 15–30% |
| **Физический товар / услуга в реальном мире** | ❌ Нет | 0% |

**SportOS = регистрация на офлайн-мероприятие = физическая услуга → IAP не требуется.**

> [!IMPORTANT]
> Apple App Store Review Guidelines, §3.1.3(e): «Apps may facilitate approved virtual currencies and in-app purchases for physical goods and services.» Регистрация на спортивное соревнование — это услуга в реальном мире.

---

## 2. Классификация способов оплаты

### Текущая модель (v1) — «Реквизиты + чек»

```
Участник → видит реквизиты → переводит вне приложения → загружает фото чека → админ подтверждает
```

| Параметр | Значение |
|---|---|
| Проходит ли платёж через приложение? | ❌ Нет |
| Нужен ли IAP? | ❌ Нет |
| Риск отклонения Apple? | 🟢 Минимальный |
| Аналоги | Объявления на Avito, фрилансеры на FL.ru |

### Будущая модель (v2) — «СБП / SberPay / Tinkoff Pay»

```
Участник → нажимает «Оплатить» → deeplink в банковское приложение → оплата по СБП/SberPay → callback → статус обновлён
```

| Параметр | Значение |
|---|---|
| Проходит ли платёж через приложение? | ⚠️ Формально да (deeplink), фактически — в банковском приложении |
| Нужен ли IAP? | ❌ Нет — оплата за физическое мероприятие |
| Комиссия Apple/Google? | 0% |
| Комиссия банка | СБП: 0.4–0.7%, Эквайринг: 1.5–2.5% |
| Риск отклонения Apple? | 🟢 Низкий (Uber, Delivery Club, Aviasales — аналогичная схема) |
| Аналоги | Uber, Яндекс Go, Delivery Club, OZON, Wildberries |

### Будущая модель (v3) — «Онлайн-эквайринг (Tinkoff Acquiring, YooKassa)»

```
Участник → нажимает «Оплатить» → WebView/Safari → форма карты → 3D-Secure → callback → статус обновлён
```

| Параметр | Значение |
|---|---|
| Нужен ли IAP? | ❌ Нет — физическое мероприятие |
| Комиссия Apple/Google? | 0% |
| Комиссия эквайера | 1.5–3.5% |
| Рекомендация | Открывать форму оплаты через **Safari/Chrome** (не WebView) — дополнительная защита |
| Аналоги | Eventbrite, Timepad, reg.ru |

## 3. Что НЕЛЬЗЯ делать без IAP

| Сценарий | Нужен IAP? | Почему |
|---|---|---|
| Подписка «Организатор PRO» (расш. функции приложения) | ✅ Да | Доступ к функционалу |
| Разблокировка аналитики для атлета | ✅ Да | Цифровой контент |
| Покупка «монет» / кредитов внутри приложения | ✅ Да | Виртуальная валюта |
| Платный доступ к видео тренировок | ✅ Да | Цифровой контент |
| Донаты организатору | ⚠️ Зависит | Через IAP, если in-app |

## 4. Рекомендации для SportOS

### Сейчас (MVP)

- ✅ **Модель «реквизиты + чек»** — максимально безопасна
- ✅ Приложение не проводит транзакции
- ✅ В описании для App Store указать: «инструмент управления офлайн-соревнованиями»

### Ближайшее будущее

- ✅ **СБП / SberPay / Tinkoff Pay** — безопасно для регистрации на мероприятия
- ✅ Реализовать через deeplink в банковское приложение (не встроенный процессинг)
- ✅ Можно интегрировать Tinkoff Acquiring / YooKassa — комиссия только банковская

### При монетизации SaaS

- ⚠️ Подписки организатора — **через IAP** (StoreKit / Google Play Billing)
- 💡 Альтернатива: продавать подписки **только через веб-сайт** (как Spotify)
- 💡 Использовать **External Link Account Entitlement** (Apple, с 2022) — ссылка на внешний сайт для оплаты

## 5. Сравнение комиссий

| Способ | Комиссия | Кто берёт |
|---|---|---|
| IAP (Apple/Google) | 15–30% | Apple / Google |
| СБП (банк → банк) | 0.4–0.7% | Банк |
| SberPay / Tinkoff Pay | ~1.5% | Банк-эквайер |
| Tinkoff Acquiring | 1.5–2.9% | Tinkoff |
| YooKassa | 2.8–3.5% | ЮKassa |
| Перевод по реквизитам | 0% | Бесплатно |

---

## 6. Формулировка для App Store Review

При подаче приложения в App Store / Google Play, в описании указать:

> «SportOS — инструмент для организации и управления спортивными мероприятиями офлайн. Приложение позволяет организаторам управлять регистрацией, хронометражем и результатами соревнований. Оплата регистрационных взносов за участие в физических спортивных мероприятиях происходит вне приложения через банковские реквизиты организатора.»

В App Store Review Notes дополнительно:

> «This app facilitates registration for real-world sporting events (sled dog racing, canicross, skiing). All payments are for physical event participation and are processed outside the app via bank transfers. No digital goods or services are sold within the app.»
