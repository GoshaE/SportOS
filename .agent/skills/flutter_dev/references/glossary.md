# Глоссарий SportOS

Словарь предметной области. Обязателен к прочтению для новых разработчиков.

## Участники и регистрация

| Термин | Определение |
|---|---|
| **BIB** (нагрудник) | Уникальный номер участника на соревновании. Печатается на нагруднике. Может не совпадать с ID в системе. |
| **Чек-ин** | Регистрация прибытия участника на место проведения в день соревнований. |
| **Мандатная комиссия** | Проверка допуска: документы, возраст, допуск врача, страховка. |
| **Ветконтроль** | Ветеринарный осмотр животного перед стартом (обязателен в ездовом спорте). |
| **DNS** (Did Not Start) | Участник зарегистрирован, но не вышел на старт. |
| **DNF** (Did Not Finish) | Участник начал дистанцию, но не финишировал. |
| **DSQ** (Disqualified) | Дисквалифицирован за нарушение правил. |

## Дисциплины (ездовой спорт)

| Термин | Определение |
|---|---|
| **Скиджоринг** | Лыжник + 1-2 собаки. Собака тянет, лыжник помогает палками. |
| **Каникросс** | Бег + 1 собака. Собака на поводке с поясом. |
| **Байкджоринг** | Велосипед + 1-2 собаки. |
| **Картинг** | Колёсная тележка + 2-8 собак. |
| **Пулка** | Лыжные гонки + 1-2 собаки + сани-пулка с грузом. |
| **Упряжка** | Нарты + 4+ собак. Классика ездового спорта. |

## Хронометраж

| Термин | Определение |
|---|---|
| **Стартёр** | Судья на старте. Выпускает участников с интервалом. |
| **Маршал** | Судья на трассе (чекпоинт). Фиксирует прохождение. |
| **Финишный судья** | Фиксирует время пересечения финишной линии. |
| **Split (сплит)** | Промежуточное время на контрольной точке. |
| **Масс-старт** | Все участники стартуют одновременно. |
| **Раздельный старт** | Участники стартуют по одному через интервал (обычно 30 сек). |
| **Стартовый интервал** | Время между стартами участников (15/30/60 сек). |
| **Мастер-Нода** | Главное устройство хронометража (обычно на финише). Источник эталонного времени. |
| **Time Delta (Δt)** | Разница между часами двух устройств. Должна быть < 50мс. |

## Организация

| Термин | Определение |
|---|---|
| **Ops-режим** | Режим судьи/организатора. Отдельный UI с 4 вкладками. |
| **Красный флаг** | Экстренная остановка гонки. |
| **Протест** | Официальная жалоба участника на результат или решение судьи. |
| **Жеребьёвка** | Определение порядка старта (случайно или по рейтингу). |
| **Стартовый лист** | Утверждённый порядок выхода на старт. |
| **Протокол** | Официальный документ с результатами после гонки. |
| **Серия / Кубок** | Несколько этапов (мероприятий), объединённых общим зачётом. |

## Технические термины

| Термин | Определение |
|---|---|
| **Drift** | Библиотека для типобезопасной работы с SQLite в Flutter. |
| **Supabase** | Open-source backend (Auth + Postgres + Realtime + Storage). |
| **CRDT** | Conflict-free Replicated Data Type — структура для merge без конфликтов. |
| **Sync Queue** | Очередь записей, ожидающих отправки на сервер (offline-first). |
| **ShellRoute** | GoRouter: маршрут-обёртка, добавляющий общий UI (напр. bottom navigation). |
| **Provider** | Riverpod: единица состояния/данных. Аналог "сервиса" в DI. |
| **autoDispose** | Riverpod: автоматическое уничтожение провайдера когда нет подписчиков. |
