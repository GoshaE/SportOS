# SportOS AI Agent Skills Index

Этот файл содержит список всех доступных навыков (skills) для AI агента в проекте SportOS. Эти инструкции обязательны к прочтению и соблюдению в соответствующих ситуациях.

## Доступные навыки

### 1. [brainstorming](brainstorming/SKILL.md)
Используется перед началом любой креативной работы — создания новых фич, компонентов или изменения поведения. Помогает исследовать намерения пользователя, требования и архитектуру до написания кода.

### 2. [flutter_dev](flutter_dev/SKILL.md)
Скилл для разработки Flutter-приложения SportOS. Используйте при написании кода, создании фич, ревью и настройке архитектуры. Включает правила именования, структуру слоев (data/domain/presentation), стандарты Git, и технологический стек (Riverpod, Drift, Supabase, Dio).

### 3. [github_publication](github_publication/SKILL.md)
Правила публикации кода и деплоя Web-версии SportOS через GitHub Actions. Обязателен к прочтению перед выполнением `git push`. Содержит жесткие требования к `flutter analyze` и нулевому уровню предупреждений линтера для обеспечения успешной сборки на CI/CD.

*Примечание: Если вам нужно добавить новый навык, создайте папку с названием навыка, положите в нее файл `SKILL.md` со структурированным описанием и обновите этот `INDEX.md`.*
